# 主从单向同步
修改my.ini文件 
主：
```
server-id = 1
log-bin= #同步事件的日志记录文件
binlog-do-db=sunpowerdb #需要备份的数据库名，如果备份多个数据库，重复设置这个选项即可
```
从：
```
server-id=3 #（配置多个从服务器时依次设置id号）
master-host=192.168.1.2 #主服务器
master-user=root #主服务器授权的用户名
master-password=123456 #主服务器授权的密码
master-port=3306 #主服务器的端口
replicate-rewrite-db=sunpowerdb->solarman #要将主服务器的数据库同步到从服务器的指定数据库中
replicate-do-table=solarman.invertors#需要备份的数据库名，如果备份多个数据库，重复设置这个选项即可
replicate-do-table=solarman.historytieminvertors
replicate-do-table=solarman.realtieminvertors
replicate-do-table=solarman.alarinvertors
```
